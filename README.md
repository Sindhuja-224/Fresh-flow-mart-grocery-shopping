Fresh flow mart is a online grocery shopping it is completely user friendly .
 It allows users to browse various categories like vegetables, fruits, juices, and meats, manage a shopping cart, and simulate a checkout process.
 
 Register and Login: Create a personal account to start shopping.
Browse Categories: Easily find Vegetables, Juices, Meats, and Fruits.
Manage a Shopping Cart: Add items to a cart, adjust quantities, and see the price change instantly.
Checkout: Review a final order summary and receive an order confirmation.
